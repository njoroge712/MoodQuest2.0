# MoodQuest2.0
A gamified mental wellness platform that turns emotional growth into an adventure. MoodQuest helps teens build emotional resilience through storytelling, CBT-inspired quests, mood tracking, and playful micro‑activities.

## About the Project

MoodQuest 2.0 is a responsive multi-section landing page introducing the MoodQuest platform. It explains the mission, showcases the features, and presents the team behind the idea.

This project demonstrates:

Beautiful UI design

Multi-section responsive layout

Carousel interactions

FAQ accordion functionality

Clean component-based CSS structure

A polished marketing experience for a wellness app

## Features
### Hero Section

Engaging introduction with call-to-action buttons.

### Story Section

Alternating text + images describing the origin of MoodQuest.

### Product Carousel

Showcasing:
-Story Quests
-Mood Mapping
-Mini Games

### Testimonials

Feedback from teens and parents.

### Mission & Values

MoodQuest's core beliefs: Empowering, Playful, Safe.

### FAQ Accordion

Expandable Q&A system.

### Team Section

Showcasing the four main contributors.

### Contact Form

Basic form for user inquiries.
